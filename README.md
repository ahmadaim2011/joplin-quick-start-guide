# Joplin Quick Start Guide for Students

This repository contains a student-friendly Quick Start Guide for the
Joplin note-taking application.

## About the Guide
The guide introduces students to:
- Organizing notes in notebooks
- Handwritten notes
- Task lists and reminders
- Multimedia notes
- Basic collaboration
